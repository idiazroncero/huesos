Menu

  • Docs
  • HTML
  • Assets

An horizontal menu is a component based on an extended <nav> or .nav.

Unlike the nav, which is a bare-bones item, the menu includes highlighting of links and a set of decoration items.

It is expected that all items on a menu are linked. Empty links can be faked by adding a <span> item and combined with the .disabled class in order to indicate a deactivated link

Submenus are handled up to the 3rd level. If you need more levels… you might want to rethink your menu.

Menu configuration

This menu is very opinionated and intented as a quick way to bootstrap a default-looking menu. If you need something more complex or very different, you should better develop your own solution.

You can config the $menu-paddingof every menu item. Use $menu-kill-links to kill all links (defaults to true) and set a $menu-background-colorthat will be used to make all color variations. Use $menu-color-ratio to set a % that will be applied to lighten() the color on every subsequent level.

Quick tip

If you don’t want to set the default .nav values, just delete the class! The .menu styling is not dependent on the nav.

It is though recommended to use the .nav class to keep all navigation styles unified.

Example

<section>
    <h3>Simple menu</h3>
    <nav class="menu">
        <ul>
            <li><a href="#">Home</a></li>
            <li>
                <a href="#">Section</a>
                <ul>
                    <li><a href="#">Home</a></li>
                    <li><a href="#">Section</a></li>
                    <li>
                        <a href="#">Works</a>
                        <ul>
                            <li><a href="#">Home</a></li>
                            <li><a href="#">Section</a></li>
                            <li><a href="#">Works</a></li>
                            <li><a href="#">About us</a></li>
                        </ul>
                    </li>
                    <li><a href="#">About us</a></li>
                </ul>
            </li>
            <li><a href="#">Works</a></li>
            <li><a href="#">About us</a></li>
            <li><span>Contact</span></li>
            <li><span class="disabled">Exit</span></li>
        </ul>
    </nav>
</section>

    @include inline-list();
    @if $menu-kill-links {
      a {
        @include kill-link-decoration();
      }

    ul > li {
      margin:0;
      position:relative;
      z-index:1;
      > a {
        text-align:left; // Override any .right class
        display:block;
        padding:$menu-padding;
        white-space:nowrap;
        @if $menu-kill-links {
          @include set-link(
            get-color(text, primary),
            get-color(text, primary),
            contrast-color($menu-background-color),
            get-color(text, primary),
            false,
            false
          )
        }
      }
      &:hover,
      &:focus {
        > a {
            color: contrast-color($menu-background-color);
            background-color: $menu-background-color;
        }
      }
      > span {
        padding:$menu-padding;
      }
      // Second-level
      > ul {
        display:none;
        position:absolute;
        top:100%;
        left:0;
        z-index:2;
        min-width:120%;
        > li {
          display:block;
          position:relative;
          > a {
            display:block;
            padding:$menu-padding / 2 $menu-padding;
            white-space:nowrap;

            background-color: lighten($menu-background-color, $menu-color-ratio);
            @if $menu-kill-links {
              @include kill-link(contrast-color(lighten($menu-background-color, $menu-color-ratio)));
            }
            @include ritmo-font-size(0.8 * $nav-font-size);
            &:hover,
            &:focus {
              @include background-contrast(shade($menu-background-color, $menu-color-ratio));
            }
          }
        }
      }
      // Third-level
      > ul > li {
        > ul {
          display:none;
          position:absolute;
          top:0;
          left:100%;
          min-width:100%;
          z-index:3;
          li {
            a {
              background-color: lighten(lighten($menu-background-color, $menu-color-ratio), $menu-color-ratio);
              @if $menu-kill-links {
                @include kill-link(contrast-color(lighten($menu-background-color, $menu-color-ratio), $menu-color-ratio));
              }
              @include ritmo-font-size(0.7 * $nav-font-size, 4);
              &:hover,
              &:focus {
                @include background-contrast(lighten($menu-background-color, $menu-color-ratio));
              }
            }
          }
        }
        &:hover {
          > ul {
            display:block;
          }
        }
      }
      &:hover {
        > ul {
          display:block;
          > li {
            display:block;
          }
