shade($color, $dilution)

Sass Function extracted from compass. Darkens the $color by mixing it with black as specified by $dilution.

Parameters

  • $color - Color to modify
  • $dilution - % of black to mix

Example

.element {
    color: shade($grey1, 25%);
}
