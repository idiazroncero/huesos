7.1.1 #Functions.Color.ContrastColor Color

Funciones que permiten escoger y manipular colores desde las paletas definidas en _config.scss

contrast-color($color, $dark, $light)

Función que devuelve un color claro o oscuro según el valor de lightness del color de fondo

Parameters:
  • $color
    Color de fondo
  • $dark
    Color oscuro para contrastar sobre claros (por defecto, la variable $contrast-dark)
  • $light
    Color claro para contrastar sobre oscuros (por defecto, la variable $contrast-light)
